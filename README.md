# TriviaGame
----------------------------------------------
Quiz created to test your knowledge on Seven Continents in fun and easy way.
This is time-based quiz, user will get 25 seconds to answer the question.
You will get question with four options, only one can be selected at a time
once user select the option, it will submit to check if this is correct or incorrect.
Message will get display for correct or incorrect.
If user haven't select any option till 25 seconds elapsed, question get Timed out and message will show up.
